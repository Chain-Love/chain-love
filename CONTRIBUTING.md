# Contributing to Chain.Love

We welcome contributions to the Chain.Love project. Please follow these guidelines:

## How to Contribute
1. Fork the repository.
2. Create a new branch for your feature or bug fix.
3. Make your changes.
4. Ensure your code adheres to the project's style guide and standards.
5. Write clear, concise commit messages.
6. Submit a pull request.

## Style Guide
- Adhere to the principles in `Project_Guiding_Document.md`.
- Ensure CSV files are correctly formatted.
- Validate JSON where applicable.

## Pull Request Guidelines
- Follow the `PULL_REQUEST_TEMPLATE.md`.
- Keep PRs focused and within the size limits.

Thank you for contributing!
