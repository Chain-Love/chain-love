## Meta files guide

This folder contains only metadata files (`categories.json` and `columns.json`).

The **authoritative guide** for adding or removing categories and columns (including how `tools/schema.json` and `meta/*` interact) lives in the root `README.md` under _"Infrastructure schema & meta guide"_.

If you need to change `meta/categories.json` or `meta/columns.json`, follow the instructions in the root `README.md` and then apply them here.
